日本の Windows Update 日を取得する

■ これは何?
日本の Windows Update 日を取得します

■ 使い方
PowerShell プロンプトで以下コマンド入力すると、今月の Windows Update 日を返します

GetWindowsUpdateDay

2017/6 とか年月を渡すと、その月の Windows Update 日を返します。

■ セットの仕方
PowerShell プロンプトで install.ps1 を実行してください

■ Uninstall 方法
uninstall.ps1 を実行して下さい
(問い合わせが来たら Enter)

■ 動作確認環境
On Windows
	5.1
	6.2.1
	7.0.0-preview.4
